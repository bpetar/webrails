In this demo you can layout tracks on the map and then run the train along the path.

Commands:

   1 - select straight rail element

   2 - select turn rail element

   r - rotate element

   c - clear scene

   p - play/stop the train animation on the tracks

   + - speed up train

   - - slow down

   ctrl + mouse press and move - rotate (left), pan (right)

   mouse wheel - zoom
